# telemetry-unifier
Python project built during Deloitte’s Technology Virtual Experience to convert IoT telemetry data from multiple JSON formats into a unified structure using data transformation, string parsing, and timestamp normalization.
# Telemetry Unifier 🚀

This project was built as part of Deloitte's Technology Virtual Experience Program.

## 📌 Overview
The goal of this project is to unify and standardize IoT telemetry data coming from different sources into a single consistent format.

## ⚙️ Features
- Cleans and processes raw telemetry data
- Converts multiple formats into a unified structure
- Improves consistency for downstream analysis

## 🛠️ Tech Used
- Python
- Data Processing
- Basic Algorithms
